# A Dollar Through the 2000s

[adtt2s.ahamedwajibu.com](https://adtt2s.ahamedwajibu.com)

## Overview

A Dollar Through the 2000s is an interactive data visualization that answers a compelling question: **If you invested $1 in the S&P 500's best-performing stock each day from 2000-2024, how much would you have today?**
